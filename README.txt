Customer Feedback System

Features:
- Voice to Text Conversion
- Sentiment Analysis
- Automatic Reply Generation
- AI Voice Reply

Technologies Used:
- Python
- Streamlit
- OpenAI Whisper
- gTTS

How to Run:

1. Install dependencies:

pip install -r requirements.txt

2. Run the application:

streamlit run app.py
